# Career Matching Website

A comprehensive career matching platform that helps users discover their ideal career path based on their interests, skills, education, work style preferences, and salary expectations.

## Features

- **30+ Career Profiles** with detailed descriptions, qualifications, and company recommendations
- **Smart Matching Algorithm** that scores careers based on user preferences
- **Employee Reviews** - Real insights from professionals in each field
- **Location-Based Job Search** - Find positions in your city
- **Direct Job Board Integration** - Links to LinkedIn, Indeed, Glassdoor, and ZipRecruiter
- **User Authentication** - Save your profile and preferences

## How to Use

1. Open `index.html` in your web browser
2. Fill out the career assessment form with your:
   - Interests (22+ categories including technology, healthcare, creative, sports, etc.)
   - Skills (technical, creative, interpersonal, analytical)
   - Education level
   - Preferred work style
   - Salary expectations
   - Location (optional)
3. Submit to see your personalized career matches
4. Explore detailed career information, top companies, and open positions

## Technologies Used

- HTML5
- CSS3 (with custom animations and responsive design)
- Vanilla JavaScript
- LocalStorage for user data persistence
